The Showcase Payment Options (icons) plugin enables you to easily display payment method icons anywhere on your WordPress website via a shortcode.

== Description ==

The Showcase Payment Options (icons) plugin is a simple, lightweight plugin designed to allow you to display visually appealing payment method icons on your website. Whether you run an e-commerce platform, a service-providing website, or an informative blog, this plugin offers a streamlined way to show the payment options you accept.

Features:
- Supports multiple payment method icons such as Visa, MasterCard, PayPal, and many more.
- Simple settings page within the WordPress admin to choose which icons you want to display.
- Shortcode support for flexible placement on pages, posts, or widgets.
- Customizable icon sizes (in pixels) for seamless integration with your site design.
- Customizable icon spacing (in pixels) for even more personalisation. 
- Option to change the order of your payment icons.

== Installation ==

1. Upload the payment-icons-plugin folder to the /wp-content/plugins/ directory, or install the plugin directly via the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Payment Icons settings page to configure which icons are displayed.
4. Optional: fill in the 'icon order' field with the 'keys' of your payment methods to change the default, alphabetical order.
4. Add the shortcode [spopm_PM] to any page, post, or widget where you want the icons to appear.

== Frequently Asked Questions ==

= Can I adjust the size of the icons? =

Yes, you can adjust the size of the payment method icons in pixels. The default is set to 65px.

= Can I adjust the spacing between the icons? =

Yes, you can adjust the spacing between the payment method icons in pixels. The default is set to 5px.

= Can I change the order of the icons? =

Yes, you can change the order of the icons. 

= What's the shape of the icons? =

You can choose between rectangle and square icons.

= What payment method icons are included? =

We currently have icons for the following: Afterpay, AliPay, Alma, Amazon Pay, American Express, Apple Pay, Bancontact, Benefit, Belfius, blik, Bitcoin, Cartes Bancaires, Cash App, Clearpay, Discover, Elo, EPS, Giftcard, Giropay, Google Pay, Grabpay, iDEAL, ING, Klarna, KBC, Knet, Maestro, Mastercard, MBway, Mercado Pago, MobilePay, Multibanco, Oxxo, PayPal, Paylib, PayU, Pix, Przelewy24, SEPA, Skrill, Sofort, Stripe, Square, UnionPay, Visa and WeChat Pay.

= Does this plugin work with every WordPress theme? =

The plugin is designed to work with most themes, but we cannot guarantee perfect compatibility due to the diversity of themes available.
